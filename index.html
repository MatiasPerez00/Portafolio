Juegos
Copia de "Mar. Bross"

Realizar una copia de uno de los juegos mas impotantes en la historia de los videojuegos. Para este proyecto se pueden implementar HTML, Css ,JavaScript para lograrlo. Podría hacerse con c++ y ciertas libreras para crear mapas con matrices, sin embargo buscamos solo darle cierta funcionalidad para mostrar las habilidades

Snake

Realizar un juego lo más parecido al mitico Snake, el juego de la "viborita" que se va alimentando de los puntos que aparecen en pantalla,agrandando su tamaño y asi lograr aumentar el nivel de dificultad. Lo podemos realizar simplemente con C++ o con algo mas visual con HTML ,Css, JavaScript

Ahorcadito

Incluso con lápiz y papel nos divertiamos con este juego. Se trata de un juego en el que uno decide una palabra y el otro tiene que ir adivinando las letras que lo conforman hasta adivinar o completar la palabra. Por cada error que se cometa se irá dibujando un muñec ahorcado. al completar el muñeco perderemos el juego. Podemos implementarlo simplemente con C++

Batalla Naval

Un jueo de barcos donde de un lado decidimos donde colocaremos unbarco en un mapa de matriz que podemos elejir la posicion de fila y columna de nuestro barco, y otra matriz para atacar a mi oponente. Si mi ataque esta en la misma posicion que el barco del contricante, significa que le di a ese barco. Se puede implementar con C++
